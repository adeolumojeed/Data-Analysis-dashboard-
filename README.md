# GreenTrail-Analysis-dashboard
GreenTrail Outdoors Promotion Performance Dashboard is a Power BI project that analyzes campaign performance using sales and marketing data. It evaluates store performance, customer behavior, seasonal trends, discounts, and ROI, providing interactive insights to support data-driven marketing decisions and improve promotional effectiveness.

## Overview

This Power BI project analyzes the effectiveness of GreenTrail Outdoors' promotional campaigns using historical sales and marketing data. The dashboard evaluates store performance, customer behavior, seasonal trends, discounts, and ROI to support data-driven marketing decisions.

## Business Problem

GreenTrail Outdoors experienced inconsistent campaign results across store locations and promotional periods. The objective was to identify the strategies that improve sales, customer engagement, and profitability.

## Project Objectives

* Analyze promotional performance across stores.
* Evaluate customer response and seasonal trends.
* Measure the impact of discounts on sales.
* Identify opportunities to improve campaign ROI.

## Key Insights

* Promotions performed best during peak outdoor seasons.
* Urban stores generated stronger sales than other locations.
* Heavy discounts did not always improve profitability.
* Targeted campaigns were more effective than broad promotions.
* Sustainability-focused initiatives can strengthen customer loyalty.

## Dashboard Features

* Executive KPI Cards
* Store Performance Analysis
* Seasonal Sales Trends
* Promotion Performance by Location
* Customer Segment Analysis
* Interactive Filters and Drill-through

## Recommendations

* Focus on targeted rather than broad discount campaigns.
* Personalize promotions by customer segment and location.
* Use A/B testing to evaluate campaign performance.
* Track additional customer and operational metrics to improve future campaigns.

## Conclusion

This dashboard demonstrates how Power BI can transform promotional data into actionable insights, helping GreenTrail Outdoors optimize marketing strategies, improve ROI, and support sustainable business growth.
